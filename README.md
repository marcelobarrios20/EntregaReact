# Tu Mascota

## Tecnologías Utilizadas

Este proyecto se desarrolló utilizando las siguientes tecnologías y herramientas:

1. **React**: Una biblioteca de JavaScript para construir interfaces de usuario. Utilizada para crear los componentes de la aplicación y gestionar el estado.
   - **React Router**: Biblioteca para manejo de rutas en React. Permite la navegación entre diferentes vistas de la aplicación sin recargar la página.
   - **@mui/material**: Biblioteca de componentes de interfaz de usuario para React, utilizada para los componentes de diseño y estilo.
2. **Firebase**: Plataforma de desarrollo de aplicaciones de Google, utilizada para la base de datos en tiempo real y el almacenamiento de datos.

   - **Firebase Firestore**: Utilizado para almacenar y recuperar datos de productos y usuarios.
   - **Firebase Authentication**: Para la gestión de autenticación de usuarios.

3. **JavaScript**: Lenguaje de programación utilizado para desarrollar la lógica de la aplicación.

   - **Módulos ES6**: Utilizados para importar y exportar componentes y archivos JavaScript.

4. **HTML y CSS**: Utilizados para estructurar y dar estilo a la aplicación.

   - **CSS Modules**: Para un scoped styling específico de componentes.
   - **Flexbox**: Para el diseño y alineación de los elementos en la interfaz de usuario.

5. **Vite**: Un bundler rápido y eficiente utilizado para compilar y servir la aplicación durante el desarrollo.
